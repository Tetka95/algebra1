Uvod u HTML

HTML je jezik za obilježavanje koji koristimo za izradu web stranica.
Njime definiramo strukturu i sadržaj web stranica.

Povijest HTML-a

HTML je nastao početkom 1990-ih godina.

Uređivanje teksta

Tekst je najčešći sadržaj web stranica.
